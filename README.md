# mustakimshikalgar.dev

Personal portfolio — built with Next.js 14, Tailwind CSS, and Framer Motion.

## Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS + custom CSS variables
- **Animations**: CSS + React Bits-inspired particle canvas
- **Data**: GitHub API for live repo stats
- **Deployment**: Vercel

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Adding Your Resume

Place your resume PDF at `public/resume.pdf`.

## Deployment

Push to GitHub and connect to Vercel. Auto-deploys on every push to `main`.
